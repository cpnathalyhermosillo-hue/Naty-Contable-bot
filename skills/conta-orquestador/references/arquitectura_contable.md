# Arquitectura contable de referencia

- Frontend: Next.js en Vercel.
- Backend: Cloud Run + Cloud SQL + Storage + Pub/Sub.
- IA: Vertex AI/Gemini para extracción y clasificación.
- Integraciones: Gmail, Drive, Calendar, WhatsApp API, PAC.
- Seguridad: RBAC, auditoría, cifrado y segregación multi-tenant.
